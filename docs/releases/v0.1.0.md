# FearMore Project Installer Bootstrap v0.1.0

Download **FearMore-Project-Installer-Bootstrap.exe**, run it, and leave **Build and install FearMore now** selected.

The bootstrap is the simple public entry point. It helps install the required public build tools, clones this exact FearMore release with its pinned EchoPatch submodule, helps locate F.E.A.R. Public Tools 1.08, validates the selected SDK `Source` folder, builds the playable Project Installer locally, and opens that setup.

Requirements and expectations:

- a legally acquired and installed copy of F.E.A.R. v1.08;
- Windows 10 or 11 with WinGet available;
- F.E.A.R. Public Tools 1.08—first check the game's `extras\fear_publictools_108.exe`, or use the verified link shown by the bootstrap;
- several gigabytes of free space and download capacity for public build tools and dependencies;
- acceptance of Windows administrator prompts for the build tools; and
- acceptance of an unsigned-publisher warning for this community bootstrap.

The bootstrap contains no retail game files, Public Tools SDK files, HD textures, compiled FearMore game modules, or EchoPatch/dgVoodoo2/SDL gameplay binaries. `FearMore-Bootstrap-Manifest.json` records that boundary, and `SHA256SUMS.txt` verifies the Release files. The playable `FearMore-Setup.exe` is produced only on the legal owner's PC and is not a public Release asset.

If Public Tools is already installed, select its folder named `Source`, not the retail F.E.A.R. folder. Build logs are saved under `%LOCALAPPDATA%\FearMore\Bootstrap\logs`.
