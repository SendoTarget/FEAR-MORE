# FearMore Project Installer Bootstrap v0.1.2

Download **FearMore-Project-Installer-Bootstrap.exe**, run it, and leave
**Build and install FearMore now** selected.

This maintenance release adds a component-scoped public license without
changing the playable remaster behavior introduced in v0.1.1. Independently
written FearMore tooling, documentation, build scaffolding, and named new
source modules are reusable under MIT. EchoPatch modifications remain
GPL-3.0-only, while the SDK-relative patch, inherited F.E.A.R./LithTech
material, retail content, HD textures, middleware, and generated combined
binaries remain outside the FearMore license grant.

The exact allowlist and exclusions are recorded in `LICENSE.md`. The public
repository boundary test requires that notice in every candidate checkout.

The bootstrap installs or locates the required public build tools, clones this
exact FearMore release with its pinned EchoPatch submodule, helps locate
F.E.A.R. Public Tools 1.08, validates the selected SDK `Source` folder, builds
the playable Project Installer locally, and opens that setup.

Requirements and expectations:

- a legally acquired and installed copy of F.E.A.R. v1.08;
- Windows 10 or 11 with WinGet available;
- F.E.A.R. Public Tools 1.08—first check the game's
  `extras\fear_publictools_108.exe`, or use the verified link shown by the
  bootstrap;
- several gigabytes of free space and download capacity for public build tools
  and dependencies;
- acceptance of Windows administrator prompts for the build tools; and
- acceptance of an unsigned-publisher warning for this community bootstrap.

The bootstrap contains no retail game files, Public Tools SDK files, HD
textures, compiled FearMore game modules, or EchoPatch/dgVoodoo2/SDL gameplay
binaries. `FearMore-Bootstrap-Manifest.json` records that boundary, and
`SHA256SUMS.txt` verifies the release files. The playable `FearMore-Setup.exe`
is produced only on the legal owner's PC and is not a public release asset.

If Public Tools is already installed, select its folder named `Source`, not the
retail F.E.A.R. folder. Build logs are saved under
`%LOCALAPPDATA%\FearMore\Bootstrap\logs`.
