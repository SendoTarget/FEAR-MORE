FEARMORE PRIVATE OWNER BUILD - NOT A PUBLIC RELEASE

This package is a convenience copy for an owner who already possesses a legal
F.E.A.R. v1.08 installation. It contains no retail executable, retail archives,
HD texture files, saves, profiles, crash evidence, or user-specific registration.

Normal Stable/Modern launches are one-click after prerequisites are present.
Optional HD textures have one additional fresh-machine prerequisite: the owner must
use the packaged stock EchoPatch lane once to create a private, attested LAA
FEAR.exe/backup pair under %LOCALAPPDATA%\FearMore. That pair is created from
the owner's own retail executable, is never written back to the retail install,
is not bundled here, and must not be uploaded or shared. The current package
does not automate that first EchoPatch launch/restart; see QUICKSTART.md.

The rebuilt game modules inherit unresolved F.E.A.R./LithTech provenance, and
the package also contains separately licensed pinned local dependencies. Do not
upload, publish, sell, mirror, or redistribute this binary package until every
component has completed a dedicated license and source-provenance review.
